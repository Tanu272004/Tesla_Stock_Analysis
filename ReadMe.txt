# 🚗 Tesla Financial & Stock Performance Analysis (2015–2024)

A complete end-to-end data analytics project that simulates Tesla’s financial and stock data, stores it in a cloud-like environment using Azurite, analyzes it using SQL, and visualizes key business KPIs in Power BI — tailored for MAANG-level roles.

---

## 📊 Project Overview

This project mimics a real-world business scenario where a data analyst must:
- Generate & clean financial data
- Store and query it in a database
- Simulate cloud integration
- Build interactive dashboards with actionable insights

---

## 🛠️ Tools & Technologies

| Component        | Tool / Tech Used         |
|------------------|--------------------------|
| Data Generation  | Python (NumPy, Pandas)   |
| Cloud Simulation | Azurite (Azure Emulator) |
| Database         | MySQL Workbench          |
| Dashboarding     | Power BI                 |
| Query Language   | SQL, DAX                 |
| Version Control  | Git + GitHub             |
